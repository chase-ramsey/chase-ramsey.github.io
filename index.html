Broadsheet

Broadsheet requests up-to-date RSS feeds from a handful of major media outlets, parses the resulting XML documents, and displays the fetched articles in the browser. All users start with the same basic feeds, but users can also add custom feeds by submitting a simple form. AngularJS structures user interactions with the app—such as article rendering, topic filters, and user comments—and Firebase is used to store persistent application and user data.

NOTE: because Broadsheet currently only operates on the browser-side, to use the app, users will need to install a browser extension to clear up any issues with cross-origin resource sharing (CORS). Check out the Github repository linked below for more details and instructions.

Github repository

Features

General
Read the 10 latest articles posted by several major media websites, including CNN, the New York Times, Pitchfork, and The Verge
Filter all news by topic with the dropdown in the navigation bar
Use the search box to look for keywords in your article results
See only news from a specific outlet by clicking its title on any article card
Click the body of an article to bring up its spotlight view, which shows more text and images for the article (where available)
Each article title is linked to the original story—check out the full article at the outlet's website!
User
Subscribe to any of the basic feeds to site provides to get articles catered more to your interests
Create custom feeds using the RSS subscribe link from your favorite sites
Talk about stories with other users in the comment section of an article's spotlight view
Save article results for further reading later, along with the conversation happening in the comments
